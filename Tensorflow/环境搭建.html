环境搭建

环境搭建

  1. Anaconda,python3.7
    • Anaconda 集成了python3.7
    • 安装过程中需要将环境变量勾上
    • 不需要安装VSCode
  2. 在cmd中输入 conda list 判断是否安装成功
  3. CUDA
    • 必须是有NVIDIA显卡才需要安装,如果不是这种显卡,一定不能安装
  4. TensorFlow
    • pip install tensorflow-cpu==2.1.0 --ignore-installed 安装CPU版本
    • GPU版本暂不考虑
  5. PyCharm
    • 需要破解,建议安装2019.2以后的版本
